---
title: "Check Session Validity"
api: "GET /api/fitnearn/web/users/check-session-validity"
description: "Check if a user session token is valid"
---

### Query Parameters

<ParamField query="token" type="string" required>
  The JWT token to be checked for validity
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the operation was successful
</ResponseField>

<ResponseField name="message" type="string">
  A message describing the result of the operation
</ResponseField>

### Response Examples

<CodeGroup>

```json Success 200
{
  "success": true,
  "message": "Session is valid"
}
{
  "success": false,
  "message": "Token is required and must be a string"
}

</CodeGroup>
